**Parallel Motion Interpolate** — бесплатный open-source инструмент командной строки, который разбивает видео на фрагменты, параллельно интерполирует каждый из них с помощью фильтра `minterpolate` FFmpeg и склеивает результат в единый выходной файл.

---

## Зачем PMinterpolate?

**SVP (SmoothVideo Project) стал платным — даже на Linux.** Если вы привыкли наслаждаться плавным воспроизведением и не хотите платить за подписку, чтобы смотреть собственные видеофайлы, PMinterpolate — ваш ответ. Никакой DRM, никаких лицензионных ключей, никакого облака — только FFmpeg, работающий быстро.

### Как ощущается высокая частота кадров?

Большинство видео снимается при 24 или 30 fps. Глаза и мозг способны воспринимать движение значительно быстрее. Интерполяция кадров генерирует промежуточные кадры, которые камера так и не сняла:

- **60 fps** — первый порог, где движение перестаёт казаться «кино» и начинает ощущаться *настоящим*. Спорт, экшн-сцены и быстрые панорамы становятся разительно чище.
- **90 fps** — золотая середина для анимации и сериалов. Персонажи двигаются с такой плавностью, что кажется почти физической.
- **120 fps** — детали, прежде размытые смазом движения, становятся чёткими и читаемыми. Игровые записи, концерты и документальные фильмы о природе оживают.

Один раз посмотрев рендер в 120 fps того, что вы сотни раз видели при 24 fps, вы уже не сможете смотреть на оригинал без ощущения матового стекла перед глазами.

## Возможности

- **Пакетная обработка папки** — бросьте папку любого размера и уйдите. PMinterpolate сканирует её автоматически и обрабатывает каждый медиафайл по одному, пропуская уже готовые.
- **Настоящий параллелизм** — видео делится на N фрагментов, все интерполируются одновременно: по одному процессу FFmpeg на каждый фрагмент. Время обработки линейно сокращается с ростом числа ядер CPU.
- **Автоопределение CPU** — `--split` не обязателен. По умолчанию используются все доступные логические ядра.
- **Все потоки сохраняются** — перекодируется только видео. Аудиодорожки, субтитры, главы и вложения копируются из источника без изменений.
- **Любой целевой FPS** — 60, 90, 120 или любое другое значение. Вы решаете.
- **Контейнеры MKV и MP4**.
- **Однопроходный и двухпроходный** режимы интерполяции.
- **18 поддерживаемых форматов**: mkv, mp4, avi, mov, mpeg, mpg, wmv, flv, webm, ts, m2ts, mts, m4v, 3gp, ogv, vob, divx, xvid.
- **Одно окно терминала** — весь вывод всех параллельных процессов отображается в том же терминале, из которого запущена команда. Никаких всплывающих окон.
- **Кросс-платформенность** — Windows и Linux, собирается из одного файла с исходным кодом.
- **Двуязычный интерфейс** — English и Русский (`--lang ru`).

## Быстрый старт

```bash
# Обработать все медиафайлы в текущей папке, автоопределение ядер CPU
PMinterpolate.exe

# Обработать конкретную папку при 60 fps, используя 8 ядер
PMinterpolate.exe "D:\Videos\Films" --fps 60 --split 8

# Двухпроходный режим для максимального качества
PMinterpolate.exe "D:\Videos" --fps 120 --passes 2 --lang ru
```

Выходные файлы сохраняются в подпапку `output/` с именем `<оригинал>_60fps.mkv`.  
Уже конвертированные файлы автоматически пропускаются при повторном запуске.

## Требования

- [FFmpeg](https://ffmpeg.org/download.html) в переменной окружения `PATH`
- Windows 7+ или Linux

Установка не требуется. Никаких зависимостей во время выполнения. Только один исполняемый файл.
