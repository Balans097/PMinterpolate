# PMinterpolate vs. ИИ-интерполяция: сравнение подходов

> **PMinterpolate** — open-source CLI-инструмент, использующий классический алгоритм `minterpolate` из FFmpeg с параллельной обработкой по ядрам CPU.  
> **ИИ-интерполяция** — нейросетевые модели (RIFE, DAIN, FILM, TopazVideo AI и др.), предсказывающие промежуточные кадры с помощью обученных сетей.

---

## Сравнительная таблица

| Критерий | PMinterpolate (FFmpeg) | ИИ-интерполяция |
|---|---|---|
| **Стоимость** | Бесплатно, MIT-лицензия | Качественные интерполяторы всегда платные (TopazVideo, SVP); бесплатные требуют настройки и дают результаты хуже |
| **Приватность** | Полностью локально, без облака | Вне зависимости от инструмента всегда не приватно |
| **Качество на простых сценах** | Отличное | Хорошее |
| **Качество на сложных сценах** | Артефакты, «призраки» | Несколько лучше |
| **Обработка окклюзий** | Слабая | Хорошая |
| **Аппаратные требования** | Только CPU | Желателен GPU |
| **Скорость** | Линейный рост с числом ядер CPU | Быстро на GPU, медленно на CPU |
| **Пакетная обработка** | Встроена «из коробки» | Зависит от инструмента |
| **Поддержка форматов** | 18 форматов | Ограниченный набор |
| **Сохранение аудио/субтитров** | Автоматически, без перекодирования | Зависит от инструмента; часто перекодирует |
| **Простота запуска** | Один .exe + FFmpeg | Требует настройки окружения |
| **Кросс-платформенность** | Windows + Linux | Варьируется; чаще онлайн через браузер |

## Достоинства

### ✅ PMinterpolate

- **Нулевая стоимость** — открытый исходный код, лицензия MIT, никаких подписок
- **Простота развёртывания** — один исполняемый файл + FFmpeg в PATH, никаких зависимостей
- **Настоящий параллелизм** — видео обрабатывается параллельно по числу ядер CPU; время обработки линейно сокращается
- **Пакетная обработка** — сканирует папку целиком, пропускает уже готовые файлы
- **Все потоки сохраняются** — аудио, субтитры, главы и вложения копируются без перекодирования
- **18 поддерживаемых форматов** — mkv, mp4, avi, mov, webm, ts, vob и другие
- **Конфиденциальность** — никакого облака, никакой телеметрии
- **Двуязычный интерфейс** — English и Русский (`--lang ru`)

### ✅ ИИ-интерполяция

- **Выше качество на сложных сценах** — взрывы, дым, вода, быстрое движение обрабатываются несколько лучше
- **Понимание семантики** — модель «видит» объекты, что снижает двоение на перекрытиях
- **Меньше артефактов** — нейросеть реже создаёт артефакты и смазы, но **может вносить несуществующие объекты**
- **GPU-ускорение** — на современных видеокартах работает в разы быстрее CPU-алгоритмов
- **Сохранение мелких деталей** — текстуры при движении воспроизводятся точнее

---

## Недостатки

### ❌ PMinterpolate

- **Имеется потолок качества алгоритма** — `minterpolate` основан на блочном поиске; на динамичных сценах иногда возникают артефакты и двоение мелких объектов
- **Нет GPU-ускорения** — только CPU; при большом числе ядер быстро, но GPU-решения обычно выигрывают на длинном видео
- **Нет семантического понимания** — алгоритм не различает объекты, поэтому на перекрытиях качество может снижаться
- **Двухпроходный режим медленнее**, но обеспечивает лучшее соотношение качество/размер файла

### ❌ ИИ-интерполяция

- **Стоимость** — лучшие инструменты (TopazVideo AI, SVP) всегда платные
- **Требует GPU** — без дискретной видеокарты скорость на CPU-режиме крайне низкая
- **Сложность настройки** — нет такого удобного CLI с пакетной обработкой «из коробки»
- **Меньше форматов** — многие инструменты работают только с ограниченным набором контейнеров
- **Нет возможности установить кодек** — инструменты работают только с ограниченным набором кодеков; предустановки не всегда оптимальны
- **«Галлюцинации»** — нейросеть иногда генерирует детали, которых в оригинале не было

---

## Вывод

**PMinterpolate** — оптимальный выбор для большинства задач: нулевая стоимость, мгновенный старт, пакетная обработка и полная приватность. Для видео с умеренным движением результат будет отличным.

**ИИ-интерполяция** выигрывает по качеству картинки на динамичных сценах, но требует GPU, бюджета или технической экспертизы.

Оба подхода **дополняют друг друга**: PMinterpolate закрывает 95% повседневных задач без лишних усилий; ИИ-инструменты подключаются там, где требуется «видеть» объекты и «додумывать» видеоконтент.
